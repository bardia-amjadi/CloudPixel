آموزش نصب و راه‌اندازی CloudPixel

نسخه راهنما نصب CloudedPixel

در این آموزش یاد می‌گیرید چطور ابزار CloudedPixel را نصب کرده و سیستم خود را برای استفاده از سرویس‌های CloudPixel آماده کنید. این مراحل فقط چند دقیقه زمان می‌برد.

مرحله 1 — دانلود برنامه

ابتدا به صفحه انتشار نسخه‌ها در گیت‌هاب بروید و نسخه متناسب با سیستم‌عامل خود را دانلود کنید: 

https://github.com/bardia-amjadi/CloudPixel/releases

مرحله 2 — استخراج فایل‌ها

پس از دانلود، فایل را Extract کنید و فایل اجرایی cloudedpixel.exe را به مسیری دلخواه (مثلاً مسیر زیر) منتقل کنید: 

C:\CloudedPixel

مرحله 3 — نصب سیستم

برنامه را اجرا کنید و دستور زیر را وارد کنید:

install --here

حالا صبر کنید تا نصب کامل شود. در صورت استفاده از Windows 11، ممکن است برنامه یک Profile برای Shell سیستم اضافه کند.

مرحله 4 — فعال‌سازی دسترسی ریشه (Root)

پس از نصب موفق، دستور زیر را اجرا کنید:

expose --root

اگر همه‌چیز درست باشد، خروجی مشابه زیر مشاهده می‌کنید:


Name :          Clouded v2
Os :            Windows 11 Pro
Path :          C:\CloudedPixel
Services :
web-app = false
background = false

Other informations are hide for secuiry reasons

اگر این اطلاعات نمایش داده شد، یعنی سیستم شما به درستی نصب شده است.

بررسی صحت نصب

برای اطمینان از نصب صحیح، دستور زیر را اجرا کنید: 

expose --root#install-success

خروجی باید مقدار true یا false باشد. اگر true نمایش داده شود، یعنی CloudedPixel با موفقیت نصب شده است.

نکات مهم

  • حتماً برنامه را با دسترسی مناسب اجرا کنید.
  • مسیر نصب را تغییر ندهید مگر اینکه مطمئن باشید چه کاری انجام می‌دهید.
  • اطلاعات حساس سیستم عمداً نمایش داده نمی‌شوند.

اتصال سیستم به حالت آنلاین و راه‌اندازی Tunnel

راهنمای اتصال CloudPixel

در این آموزش یاد می‌گیریم چگونه سیستم خود را از حالت آفلاین به آنلاین تغییر داده و اتصال Tunnel را برای برنامه‌های خود برقرار کنیم. این مراحل به شما کمک می‌کنند تا سیستم شما بتواند به سرویس‌های ابری CloudPixel متصل شده و از امکانات مدیریت و کنترل برنامه‌ها به صورت آنلاین استفاده کند.

مرحله 1 — ورود به حساب کاربری

ابتدا دستور زیر را در کنسول وارد کنید:

login

بعد از اجرای این دستور، یک صفحه مرورگر در آدرس localpixel باز می‌شود. شما باید نام کاربری و رمز عبور خود را وارد کنید تا اتصال به حساب کاربری CloudPixel انجام شود.

پس از موفقیت، در کنسول پیام زیر را مشاهده خواهید کرد: 

You successfully logged in your account

اگر پیام موفقیت را مشاهده نکردید یا خطایی دریافت کردید، دوباره دستور login را اجرا کنید. این عملیات ممکن است در اولین بار با تأخیر انجام شود یا نیاز باشد مرورگر خود را ریفرش کنید.

مرحله 2 — اتصال به Tunnel

پس از ورود موفق به حساب، وقت آن است که سیستم خود را به یک Tunnel متصل کنیم. برای این کار دستور زیر را وارد کنید: 

link tunnel {TUNNEL_TOKEN}

پس از اجرای دستور، اطلاعاتی مشابه زیر نمایش داده می‌شود: 


Owned by {your name}
Domains : {Your Subdomain or Domains}
Last Ping : {Last Ping}
Available : true/false

This is your application? Y/N

گزینه Y را انتخاب کنید تا اتصال برنامه شما تأیید شود. سپس سیستم از شما می‌خواهد رمز عبور خود را وارد کنید: 

Please enter your secret :

بعد از وارد کردن رمز عبور، باید یک LocalName برای این برنامه تعیین کنید. این نام به شما کمک می‌کند برنامه خود را در آینده راحت‌تر مدیریت کنید:

Please set a localname for this app :

مرحله 3 — انتظار برای ایجاد اتصال

بعد از وارد کردن اطلاعات، سیستم شروع به ایجاد اتصال می‌کند. این فرایند ممکن است چند دقیقه طول بکشد و پیام‌هایی مانند زیر را مشاهده خواهید کرد: 


Collect Information From CloudPixel Edge      [Success]
Validate the Information for security reasons  [Success]
Create new instance of CoForgent                   [Success]
Including the configuration                              [Success]
Ping the web application                                  [Success] 
Shutting down the application                         [Success]

هر پیام [Success] به این معناست که آن مرحله با موفقیت انجام شده است. پس از پایان این مراحل، اتصال به Tunnel برقرار شده و سیستم شما آماده استفاده است.

مرحله 4 — استفاده از Tunnel

اکنون می‌توانید از دستورات Tunnel برای مدیریت برنامه‌ها و کنترل اتصالات استفاده کنید. نمونه دستورات:

  • tunnel <lname> --<field> --<value> برای تنظیم مقادیر مشخص
  • tunnel <lname> --funcs --<name> <args> برای اجرای توابع خاص برنامه

مثال:


tunnel myapp --incoming --3
tunnel myapp --funcs --start

برای مشاهده تمامی دستورات و راهنمایی‌ها، دستور --help را وارد کنید تا لیست کامل گزینه‌ها و قابلیت‌ها نمایش داده شود.

نکات مهم

  • دستورها را با دقت وارد کنید تا اتصال بدون مشکل برقرار شود.
  • نام LocalName برنامه را به خاطر بسپارید، زیرا برای اجرای دستورات بعدی ضروری است.
  • رمز عبور خود را محرمانه نگه دارید و با کسی به اشتراک نگذارید.
  • در صورت بروز خطا یا مشکل اتصال، یک بار سیستم را ریستارت کنید و دوباره مراحل را انجام دهید.
  • هنگام اجرای دستورات، اتصال اینترنت شما پایدار باشد تا مراحل به درستی انجام شوند.
